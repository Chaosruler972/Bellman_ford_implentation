# Bellman_ford_implentation
Simulating routers using Bellman Ford Algorithm

## To Install
	run 'make'
	
## To Run:
	running command : ./[executable file name] <input file> <Router's name> <amount of reconnect times>
	
## To Uninstall
	run 'make clear'
